# Black Shadow Game — Official Website

The official website for **Hongkong Black Shadow Game Technology Limited**.

## Live Site

https://about.blackshadowgame.com

## Tech Stack

- Static HTML + Tailwind CSS (CDN)
- Dark game-company aesthetic with neon accents (cyan, purple, pink)
- Responsive design
- Custom SVG favicon (BG shield icon)

## Project Structure

```
about.blackshadowgame.com/
├── index.html       # Homepage
├── privacy.html     # Privacy Policy
├── terms.html       # Terms of Service
└── favicon.svg      # BG shield icon
```

## Deploy

Deployed on [Cloudflare Pages](https://pages.cloudflare.com/).

```bash
# Deploy with Wrangler
wrangler pages deploy . --project-name=blackshadowgame --branch=main
```
